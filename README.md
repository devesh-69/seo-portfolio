## 💼 Case Studies

### ISMT Business School (`ismtindia.com`)

**Niche:** Education — BCA / BBA / MBA Admissions, Mumbai  
**Timeline:** March 2025 – Present

**Challenge:** New, competitive education website with near-zero organic visibility in a crowded Mumbai admissions market.

**What I Did:**

- Full technical SEO audit — fixed crawlability and indexing issues
- Built pillar–cluster content strategy around BCA, BBA, MBA keywords
- Implemented Article, FAQ, and Breadcrumb schema markup
- Optimised on-page: meta tags, headings, internal links, image alt text
- Executed backlink outreach to build domain authority

**Result:** Organic traffic share 51% → 78% · Indexing 2–3 weeks → 3–5 days · 11+ Page 1 keywords

---

### CareerManagers (`careermanagers.in`)

**Niche:** Exam Prep — CET, RRB, NABARD, GATE  
**Timeline:** 2025 – Present

**Challenge:** Existing content platform with poor discoverability for high-intent study material and mock test queries.

**What I Did:**

- Identified high-intent, low-competition study material and mock test keywords
- Restructured existing content for better search intent alignment
- Added structured data for course/exam content pages
- Improved internal linking across syllabus, study material, and mock test pages
- Built topical authority across CET, RRB, NABARD, and GATE verticals

**Result:** Organic traffic share 42% → 71% · Average position 24 → 6 · 10 keywords ranked

---

### ExamTarikh (`examtarikh.in`)

**Niche:** Government Exams — SSC CGL, SBI PO  
**Type:** Personal Project — built and ranked solo

**Challenge:** Prove SEO skills independently, outside of employment.

**What I Did:**

- Built site architecture and content strategy from scratch
- Targeted question-form and long-tail SSC/SBI keywords
- Created unique tools (SBI PO pension calculator) to earn organic traffic
- Applied full technical SEO independently — indexing, schema, on-page

**Result:** 4 Page 1 rankings including Position 1 for the pension calculator tool

## 🚀 Local Setup

```bash
# Clone
git clone https://github.com/devesh-69/seo-portfolio.git
cd seo-portfolio

# Install with Bun
bun install

# Start dev server
bun run dev
```

Open [http://localhost:5173](http://localhost:5173)
